### 메시지 출력 화면
- 아이디를 정하여 해당 아이디를 가진 사용자에게 메시지를 보낼 수 있습니다.
- 발신자와 제목을 아스키 아트와 함께 출력합니다.
- 커맨드 입력을 통해 메시지의 전송 받은시간,내용 등을 출력합니다.
